# 🧳 Living – Travel Booking App UI Kit

A mobile UI kit for a travel booking app called **Living**, designed in Figma. This project includes high-fidelity UI screens, low-fidelity wireframes, a full design system (typography, colors, shadows), and reusable components — 99+ screens in total.

## 📱 Overview

Living is a travel discovery and booking app concept that lets users explore destinations, browse curated travel content ("Popping now by view"), and search/book hotels by location, dates, and guest count.

## 📂 What's Included

- **UI Screens** — Fully designed high-fidelity mobile screens (Login, Feed/Home, Hotel Search & Results, etc.)
- **Wireframes** — Low-fidelity versions of the same screens for structural/UX reference
- **Style Guide**
  - Typography scale (League Spartan — Headline, Subtitle, Body, Button, Caption styles)
  - Color palette (Main + Wireframe variants)
  - Shadow levels (Level 1, 1.1, 2)
- **Component Library**
  - Buttons, Icons, Menus, Status Bar, Navigation Bar
  - Search Fields, Input Fields
  - Cards & List items
